Chatty 
=====================

Chatty is a ReactJS chat app that takes advantage of websockets.

### Usage

Install the dependencies for both the app and the websocket server and then start the server.

```
npm install
cd chatty_server
npm install
cd ..
npm start
open http://localhost:3000
```
